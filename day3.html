    • 6.What is the z-index in CSS?

    • The z-index property specifies the stack order of an element.
    • An element with greater stack order is always in front of an element with a lower stack order.
    • z-index property will not work if the position is static.

    8.What is box sizing?

  • The CSS box-sizing property allows us to include the padding and border in an element's total width and height.
  • By default, the width and height of an element is calculated like:
    1. width + padding + border = actual width of an element
    2. height + padding + border = actual height of an element

    9.What is animation delay?

  • The animation delay property specifies the amount of time to wait from applying the animation to an element before beginning to perform the animation.
  • The animation-delay property specifies the delay for to start of animation
  • The animation value is defined in seconds(s) or milliseconds(ms)
